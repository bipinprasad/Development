Tools

Eclipse is the development IDE of choice. It can be downloaded from the http://www.eclipse.org. Download Eclipse Kepler (4.3.1) standard edition. If you do not have Java installed on your machine, you will need to do that first. Get JDK SE 7 (aka 1.7) from http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html

Python Distribution Download python version 2.7.6 32-bit version from python.org. There is a Windows x86 MSI Installer on this page. Skip the 64-bit because of persistent issues with 64 bit third party libraries.

Python development within eclipse requires PyDev plugin. The update site for this is at http://pydev.org/updates/. Update sites can be set up within eclipse. FAQ How do I upgrade Eclipse? Eclipse Update Manager Go to the update manager (Help > Install New Software) and add: http://pydev.org/updates and follow the Eclipse steps.

Google of App Engine https://developers.google.com/appengine/docs/python/

Source Code Control: Source repository is github. URLs for various repos will be updated here. Before working on a git project, familiarize yourself with git, since it works quite differently from other source control software and is quite confusing with a lot of options. Download GitHub GUI software from http://windows.github.com. You will

  • clone an existing remote (on the github) git project
  • create and switch to a branch
  • make changes and test code to the branch (local)
  • add changed files and commit changes to branch (still locally)
  • merge the change to the official branch
  • push the changes to remote
  • Note: Each of the actions above has a multi word git command, but the GUI interface makes it simpler.
